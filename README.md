# ComunicaTech 📱💬
**Solución integral de accesibilidad y comunicación para personas con discapacidad auditiva.**

ComunicaTech es una aplicación nativa de Android diseñada para cerrar la brecha de comunicación. Integra herramientas avanzadas de procesamiento de lenguaje, visión artificial y geolocalización para ofrecer un entorno seguro y funcional a sus usuarios.

## 🚀 Características Principales
- **Modo Sordo:** Traductor visual de lengua de señas mediante tarjetas interactivas (Flashcards).
- **Mapa de Emergencia:** Localización en tiempo real de puntos seguros (Hospitales, Comisarías, Bomberos) en la ciudad de Trujillo, Perú.
- **Noticias Tech:** Integración de API externa para mantener a los usuarios informados sobre las últimas tendencias tecnológicas.
- **Escáner QR:** Funcionalidad de lectura de texto mediante códigos QR con síntesis de voz.
- **Seguridad:** Sistema de autenticación robusto con Firebase, incluyendo verificación de identidad por correo electrónico y perfiles personalizados.

## 🛠️ Stack Tecnológico
- **Lenguaje:** Kotlin
- **UI:** Jetpack Compose (Material Design 3)
- **Backend/BaaS:** Firebase (Auth, Cloud Storage, Firestore)
- **Maps:** Google Maps SDK for Android
- **Arquitectura:** MVVM (Model-View-ViewModel)
- **Librerías:** Coil (Carga de imágenes), Retrofit (Consumo de APIs), Corrutinas de Kotlin.
